# Bibliothèque de jeux vidéo

Créer les classes qui permettent de représenter :
- Jeu Video (**VideoGame**)
    - ***propriétés***: nom, description, genre, nombre de joueurs, plateforme (type de console), date de sortie, note
- Bibliothèque de jeux vidéo (**Library**)
    - ***propriétés***: liste des jeux vidéos
    - ***fonctions*** : 
        - rechercher des jeux par mot clé dans le nom et dans la description.
        - filtrer des jeux par date, ou plateforme, note, genre
- Console de jeu vidéo qui représente la plateforme (**Console, XBox, Switch, PlayStation, PC**)
    - ***propriétés***: nom, ram, cpu, état (allumé ou étein), jeu en cours
    - ***fonctions*** : allumer, éteindre, démarer jeux, arréter jeux, éjeccter jeux.
