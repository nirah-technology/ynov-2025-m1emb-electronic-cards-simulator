# Armoire de cartes électronique.

## Les objects à créer :
- l'armoire qui contient les cartes électronique.
- la carte électronique (constructeur, model, ram, cpu, espace de stokage)

## Les fonctions à créer :
- Ajouter une nouvelle carte dans l'armoire
- supprimer une carte de l'armoire
- lister les cartes de l'armoire
- trier les cartes de l'armoire par : constructeur ou ram ou cpu ou espace de stockage
- filtrer le cartes de l'armoire par : constructeur ou ram ou cpu ou espace de stockage
